# Visa Document Response Checklist

## For each requested document, identify:
- source authority
- who must request it
- whether apostille is needed
- whether translation is needed
- freshness limit (e.g. 3 months)
- whether it is easy / medium / hard to obtain

## Response structure
1. Status: requested, not denied
2. Meaning in plain language
3. Required items list
4. Action plan in order
5. Risks and mistakes to avoid

## Good habits
- Never delete old uploaded docs unless explicitly instructed.
- Keep originals and translated copies organized.
- Note where equivalents may be accepted depending on jurisdiction.
