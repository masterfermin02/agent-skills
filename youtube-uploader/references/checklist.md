# Upload Checklist

## Before upload
- final file exists
- title exists
- description exists
- privacy mode chosen
- thumbnail decision made
- file is MP4 when possible

## Recommended technical target
- container: MP4
- video codec: H.264
- audio codec: AAC
- resolution: 1080p or better

## If blocked
Common blockers:
- no YouTube auth
- no upload script/tool configured
- missing metadata
- bad file path
