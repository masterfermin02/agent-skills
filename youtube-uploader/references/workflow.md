# Workflow

## Basic path
1. Render final MP4.
2. Review title and description.
3. Confirm privacy.
4. Upload.
5. Save resulting URL or video id.

## Early versions
The first version of this workflow can be metadata-first even if the final upload automation is not fully wired yet.

## Future growth
- scheduling
- thumbnail upload
- playlist placement
- reusable templates for descriptions
