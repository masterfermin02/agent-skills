# Page Operations

## Reorder pages
Use when the user wants a different page sequence across one or more PDFs.
Prefer explicit page-number instructions.
Example asks:
- put the summary page first
- move page 5 after page 1
- combine these PDFs but put the last file first

## Extract pages
Use when the user only needs selected pages for upload or evidence.
Examples:
- extract pages 1-2
- keep only the page that shows the deposit
- split this PDF into monthly sections

## Compression / optimization
Only do this when file size matters.
Prefer a copy of the PDF, not destructive overwrite.
Warn the user that aggressive compression may reduce scan/image quality.

## Annotation mindset
For evidence workflows, visible clarity matters more than perfect typography.
If a clean highlight is hard, use obvious annotations or create a supporting appendix.
