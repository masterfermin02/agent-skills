# Visa Document Response Workflow

## Trigger
The user receives a visa/immigration email requesting documents.

## Steps
1. Parse the request and extract required documents.
2. Mark apostille/translation/recency requirements.
3. Identify which docs come from employer/company/government.
4. Build a clean checklist and response strategy.
5. Explain blockers and mistakes to avoid.

## Output
- plain-language explanation
- exact requirements
- action plan
- risks
