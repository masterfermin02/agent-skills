# Rental Finder Workflow

## Trigger
The user wants an apartment or monthly stay.

## Steps
1. Clarify city, dates, budget, solo/shared, whole place/room, key priorities.
2. Search platform(s): Airbnb first if short-term furnished stay makes sense.
3. Shortlist options using budget + trust + neighborhood + work/gym fit.
4. Rank the options and call out red flags.
5. Recommend next actions.

## Output
- best fit
- budget pick
- safest pick
- tradeoffs
